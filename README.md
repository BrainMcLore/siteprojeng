# English for IT

Static class project website for **English for IT**.

## Files

- `index.html` - page structure
- `styles.css` - layout, colors, transitions, and responsive styles
- `script.js` - glossary rendering, animated term marquee, reveal effects, and background stream
- `report.md` - short written report for class

## GitHub Pages

This project is ready for GitHub Pages because it is a plain static website.

### Deploy steps

1. Create a GitHub repository.
2. Upload all project files to the repository root.
3. Open `Settings -> Pages`.
4. In **Build and deployment**, choose:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` and folder `/root`
5. Save and wait a minute.
6. GitHub will publish the site with a public URL.
