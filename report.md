# English for IT

## Project idea
This website presents the topic **English for IT** through an immersive cyber-stream design. The goal is to combine English writing practice with basic knowledge about computers and information technology.

## Main content
The website includes four short sections:

1. **What are computers?**
   Computers are electronic devices that accept data, process information, and produce results.

2. **How do they work?**
   They follow the input, processing, storage, and output model.

3. **What is the most important part?**
   The CPU is often considered the most important part because it works like the brain of the computer.

4. **What are the components?**
   The website presents the CPU, motherboard, RAM, storage, GPU, power supply, cooling, and input/output devices.

## Glossary
The project contains **24 IT terms** with short definitions and visual icon labels. This helps students connect technical vocabulary with meaning and visual memory.

## Design concept
The visual style is based on a **cyber stream** concept. The moving video, glowing colors, and liquid transitions represent data flowing through a digital system. This makes the website feel modern, creative, and more engaging for an audience.

## Presentation plan
The class presentation can follow this order:

1. Introduce the theme "English for IT".
2. Explain what computers are.
3. Show how computers work.
4. Present the CPU as the core part.
5. Describe the main components.
6. Use the glossary to explain key IT words.
7. Finish by saying why English is important in the world of technology.

## Conclusion
This project mixes language learning, IT knowledge, and modern web design. It is complete, visually creative, and suitable for a 5-10 minute presentation.
